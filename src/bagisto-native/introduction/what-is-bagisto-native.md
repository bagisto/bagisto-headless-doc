# What is Bagisto Native?

Bagisto Native is a framework that enables you to build **native mobile applications** using your existing **Next.js/React.js web projects** through **Hotwire** technology.

## Key Points

- Transforms web applications into native mobile apps
- Uses Hotwire Bridge technology for web â†” native communication
- Eliminates the need to build separate native apps from scratch
- Maintains a single codebase for web and mobile platforms
- Provides seamless integration between web components and native device features
