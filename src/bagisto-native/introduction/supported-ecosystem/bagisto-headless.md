# Bagisto Headless

Bagisto Native provides first-class support for **Bagisto Headless Commerce** - a modern, API-first e-commerce platform.

## Quick Start

Create a new Bagisto Headless storefront:
```bash
npx -y @bagisto-headless/create your-storefront
```

## Key Features

- Pre-built commerce functionality (products, cart, checkout)
- Optimized integration with Bagisto Native components
- Ready-to-use e-commerce workflows
- Minimal configuration required

## Why Use Bagisto Headless?

- **Faster Development** - Pre-built commerce features
- **Native-Ready** - Designed to work with Bagisto Native
- **Scalable** - Grows with your business
- **Flexible** - Fully customizable

## Getting Started

1. Create your storefront with the command above
2. Install Bagisto Native modules
3. Add the Hotwire bundle to your project
4. Start integrating native components
