# Any API-Driven Backend

Bagisto Native is **backend-agnostic** and can integrate with any API-driven system.

## Supported Backend Types

### REST APIs
- Standard HTTP methods
- JSON/XML responses
- Token-based authentication

### GraphQL APIs
- Query and mutation support
- Real-time subscriptions
- Flexible data fetching

### Custom Services
- Proprietary APIs
- Legacy systems with API layers
- Microservices architecture

### Third-Party Platforms
- Shopify, WooCommerce, Magento
- commercetools
- Any e-commerce or business platform

## How It Works

1. Your web app connects to your backend APIs
2. Bagisto Native adds native device features

4. Your backend remains unchanged

## Benefits

- **Freedom of Choice** - Use any backend technology
- **No Vendor Lock-in** - Not tied to specific platforms
- **Incremental Adoption** - Add features gradually
- **Flexibility** - Integrate multiple backends if needed
