# When to Use Bagisto Native

## You Have an Existing Web Application

- Already built a web application using JavaScript frameworks
- Want to extend your web app to mobile platforms
- Need to maintain feature parity across web and mobile

## You Want to Save Development Time

- Don't want to build separate native apps from scratch
- Need to launch mobile apps quickly
- Have limited resources or budget for native development

## You Need Native Device Features

- Require access to device camera for barcode/image scanning
- Need location services for address auto-fill
- Want native notifications and device-specific functionality

## You're Building an E-commerce Application

- Need mobile shopping experiences with native features
- Want product sharing, search, and cart functionality
- Require seamless checkout with location-based services

## You Want to Maintain a Single Codebase

- Prefer unified logic across all platforms
- Want easier maintenance and updates
- Need consistent user experience on web and mobile

## You're Using Hotwire Technology

- Already familiar with or using Hotwire/Turbo
- Want to leverage Hotwire's native bridge capabilities
- Prefer progressive enhancement approach
