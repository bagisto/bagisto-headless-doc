# Supported Ecosystems

Bagisto Native is designed to work with various backend systems, providing flexibility in how you build your native applications.

## Backend Support

Bagisto Native supports two main backend approaches:

### Bagisto Headless

A pre-built, API-first e-commerce platform optimized for Bagisto Native integration.

- Quick setup with ready-to-use commerce features
- Pre-configured API endpoints
- Optimized e-commerce workflows

[Learn more →](/bagisto-native/introduction/supported-ecosystem/bagisto-headless.md)

### Any API-Driven Backend

Connect to any backend system that provides APIs.

- REST APIs
- GraphQL APIs
- Custom backend services
- Third-party platforms

[Learn more →](/bagisto-native/introduction/supported-ecosystem/any-api-driven-backend.md)

## Frontend Frameworks

Works with any JavaScript framework:

- Next.js (v15+)
- React.js (v18+)
- Vue.js, Angular, Svelte
- Any framework supporting Web Components
