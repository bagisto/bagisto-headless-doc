# Configure & Run Headless Storefront

Once you have generated your Bagisto Headless project, you need to configure it to talk to your specific Bagisto Backend.

## Configuration Steps

1.  **Environment Variables**: Set up the connection strings.
2.  **Development Server**: Start the local Next.js server.
3.  **Connectivity Check**: Verify data is flowing correctly.

::: warning API Requirement
Ensure your backend GraphQL API is accessible from the machine running this storefront.
:::
