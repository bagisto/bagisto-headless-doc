# Create Bagisto Headless Commerce

The foundation of your native app is the **Bagisto Headless Storefront**. This is a modern Next.js application designed to be fast, SEO-friendly, and capable of acting as the UI for your native mobile apps.

## What you will do in this section

1.  Understand the Headless Concept.
2.  Run the CLI command to generate your project.
3.  Explore the generated project structure.

::: info Next.js Powered
Bagisto Headless is built on **Next.js**, giving you the best of React Server Components, Static Site Generation, and specific optimization features out of the box.
:::
