# System Requirements

Before diving into Bagisto Native, ensure your development environment meets the necessary criteria to build both the web frontend and the native mobile applications.

## General Requirements

*   **Operating System**: 
    *   **macOS**: Required for iOS development (Xcode).
    *   **Linux/Windows**: Suitable for Android and Web development only.
*   **RAM**: Minimum 8GB (16GB recommended) to run Android Studio/Xcode simulators smoothly.
*   **Storage**: SSD with at least 20GB free space for SDKs and emulators.

## Component Specifics

### 1. Web Frontend (Next.js)
*   Can be developed on any OS (macOS, Windows, Linux).
*   Requires a modern terminal (Terminal, PowerShell, or WSL).

### 2. Native iOS App
*   **Hardware**: Mac computer with Apple Silicon (M1/M2/M3) or Intel processor.
*   **OS**: macOS Sonoma (14.0) or later recommended.

### 3. Native Android App
*   **Hardware**: Setup capable of running Android Studio.
*   **OS**: Windows, macOS, or Linux.
